<h3>Brief role description:</h3>
<blockquote>
  I am a native <b>iOS Engineer</b> who plays a key role in developing high-performance mobile applications. My main task is to create <b>scalable, systems-oriented architectures</b> using Swift & SwiftUI, integrate real-time capabilities (WebRTC), and handle complex state management. I value correctness, performance, and a polished user experience.
</blockquote>

<br>

<h3>What I do (Experience & Projects):</h3>
<ul>
  <li><b>EnglishCallingApp:</b> Built a real-time P2P audio calling platform using WebRTC & Node.js signaling.</li>
  <li><b>Ludo (iOS Game):</b> Designed a deterministic game engine with complex animation states in SwiftUI.</li>
  <li><b>TicTacToe Multiplayer:</b> Implemented Minimax AI and fault-tolerant real-time sync via Google Cloud.</li>
  <li><b>Systems Integration:</b> Bridged C++ components into iOS via Objective-C++ for performance critical tasks.</li>
  <li><b>Architectural Design:</b> Applied MVVM and unidirectional flow to ensure testability and scalability.</li>
</ul>

<br>

<h3>Engineering Approach:</h3>
<ul>
  <li>Prioritize correctness first, performance second, and polish through iteration.</li>
  <li>Strong knowledge of <b>Swift Concurrency</b> and structured async patterns.</li>
  <li>Experience integrating low-level systems where required.</li>
  <li>Adherence to Apple's <b>Human Interface Guidelines</b> for UI implementation.</li>
</ul>
